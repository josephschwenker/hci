Research Task Instructions

In this research study, you will be asked to alphabetize several lists of words. The words will be displayed on the left, and five clickable buttons will be displayed on the right. You can use the five clickable buttons to alphabetize each list.

Do not use your web browser's back button or any other web browser buttons during the study. You may only use your left mouse button and the keyboard shortcuts provided to you, if any.

Words that would appear earlier in the dictionary should be placed toward the top of the list, while words that would appear later in the dictionary should be placed toward the bottom.

The image on the right is an example of what the interface will look like.

Select Mode or "s"

The alphabetization task will begin in select mode. You can also switch to select mode by clicking the button labeled “s.” You will know the system is in select mode when the button with an “s” has a black background and the letter “s” is white. In the picture on the right, the system is in select mode.

The word that is currently selected is marked on the left with a black background and white text. This is called the cursor. In the picture on the right, the cursor is on the word "apospories."

Available Commands in Select Mode

Click on the button labeled “^” to move the cursor up by one. If the cursor is already at the top, it will wrap around and select the word on the very bottom.

Click on the button labeled “v” to move the cursor down by one. If the cursor is already at the bottom, it will wrap around and select the word at the very top.

Move Mode or "m"

To move a word up, click on the button labeled "m" to switch to move mode. The system is in move mode when the button labeled with an m has a black background and the m is white.

In move mode, clicking on the buttons labeled "^" and "v" will move the cursor in the same direction while bringing with it the word that is currently selected. The cursor and word will also wrap around if the cursor is at the very top or the very bottom.

You can switch back to select mode at any time by clicking the button labeled s.

Occasional Keyboard Shortcuts

Sometimes, you will be able to press the letter on your keyboard that is displayed on each button, and this will have the same effect as clicking the button. For the "^" button, use the up arrow key on your keyboard, and for the "v" button, use the down arrow key on your keyboard. The study will tell you when you can do this in the instructions before the task begins.

Done Mode or "d"

When you are done alphabetizing the list, click on the button labeled with a "d." If this list is not alphabetized, an error will be displayed, and you will not be able to move on to the next task until you finish.
